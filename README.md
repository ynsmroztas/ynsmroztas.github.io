# ynsmroztas.github.io — v2.1

Security researcher portfolio built with **Astro** + GitHub Actions auto-deploy.

## Features
- Matrix rain animated background
- Dark/Light theme toggle
- Typewriter hero animation
- Hacker-style visitor counter (IP + geolocation + ISP)
- GitHub repos auto-fetched & shuffled on every visit
- CVE / Vulnerability timeline
- Scroll-triggered animations
- Blog section (Markdown-based)
- Fully responsive

## Quick Start
```bash
npm install && npm run dev
```

## Adding Blog Posts
Create `.md` files in `src/content/blog/`

## Deploy
Automatic via GitHub Actions on push to `main`.
