nextrce
Python
Automated scanner designed to detect Remote Code Execution vulnerabilities in Next.js applications. Leverages server-side rendering misconfiguration patterns, prototype pollution chains, and deserialization flaws to identify critical attack surfaces in modern Node.js web apps. Built for bug bounty hunters targeting Next.js-powered targets at scale.
RCE Next.js Scanner Automation Bug Bounty
$ python3 nextrce.py -u https://target.com
[+] Detecting Next.js version...
[!] RCE vector found — SSR deserialization
🔍
GitHub ↗
inspectjs
Python
Deep JavaScript analysis tool that extracts sensitive information, hidden endpoints, API keys, and security-relevant patterns from JS bundles. Parses minified and obfuscated JavaScript to surface secrets, hardcoded credentials, internal routing logic, and potential injection points — all from a single script file or URL.
JS Analysis Recon Secret Detection OSINT API Keys
$ python3 inspectjs.py --url https://target.com/app.js
[★] API_KEY leaked: sk-prod-xxxxxxxxxxxx
[★] Endpoint found: /api/v2/admin/users
🤖
GitHub ↗
androuhunter
JavaScript / Node.js
Automated Android application security testing framework. Decompiles APKs, hunts for exported components without proper permission checks, insecure data storage patterns, hardcoded secrets in resources, and deep link abuse vectors. Designed to accelerate mobile bug bounty reconnaissance and static analysis workflows.
Android APK Analysis Mobile Security Static Analysis Deep Links
$ node androuhunter.js --apk target.apk
[!] Exported Activity: com.target.DeepLinkActivity
[!] Hardcoded secret in res/values/strings.xml
